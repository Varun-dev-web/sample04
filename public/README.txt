V4 TECHNOLOGIES WEBSITE - PDF TEMPLATES
=======================================

IMPORTANT: Please add the following PDF files to this public folder:

1. label.pdf - Sample label template for customers to download
2. sticker.pdf - Sample sticker template for customers to download

Once you add these files, the download buttons on the website will work.

The files should be placed directly in the 'public' folder:
- public/label.pdf
- public/sticker.pdf

When you deploy to GitHub Pages or any hosting:
- These files will be accessible at /label.pdf and /sticker.pdf
